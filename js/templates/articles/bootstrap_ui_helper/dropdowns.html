Dropdown

Use the dropdown helper to display Bootstrap-styled dropdowns.

Basic dropdowns

Use dropdown with dropdown button text and dropdown list items details to create a basic dropdown.

There are 3 list item types: header, link and divider.

header: only text option is required.

link: link text and path(link) are required.

divider: nothing is required.

  
    <%= dropdown 'Dropdown', [{type: :header, text: 'Dropdown header'},
                              {type: :link, text: 'Action', link: '/action'},
                              {type: :link, text: 'Another action', link: '/another_action'},
                              {type: :divider},
                              {type: :header, text: 'Another dropdown header'},
                              {type: :link, text: 'Separated link', link: '/separated_link'}] %>

generates the following HTML:

EXAMPLE
  • Dropdown header
  • Action
  • Another action
  • Another dropdown header
  • Separated link 
  
    <div class="btn-group dropdown">
      <button name="button" type="submit" class="dropdown-toggle  btn btn-default" id="dropdown-44001d"
              data-toggle="dropdown" aria-haspopup="true" aria-expended="false">Dropdown <span class="caret"></span>
      </button>
      <ul class="dropdown-menu dropdown-menu-left" role="menu" aria-labelledby="dropdown-44001d">
        <li class="dropdown-header">Dropdown header</li>
        <li>
          <a role="menuitem" tabindex="-1" href="/action">Action</a>
        </li>
        <li>
          <a role="menuitem" tabindex="-1" href="/another_action">Another action</a>
        </li>
        <li class="divider" role="separator"></li>
        <li class="dropdown-header">Another dropdown header</li>
        <li>
          <a role="menuitem" tabindex="-1" href="/separated_link">Separated link</a>
        </li>
      </ul>
    </div>

List item options

Simply put options into the list item hash to add options for it.

  
    <%= dropdown 'Dropdown',
                  [{type: :link, text: 'Action', link: '/action', class: 'action', data: {js: 1}},
                   {type: :link, text: 'Another action', link: '/another_action', class: 'another-action', data: {js: 2}}] %>

generates the following HTML:

EXAMPLE
  • Action
  • Another action 
  
    <div class="btn-group dropdown">
      <button name="button" type="button" id="dropdown-144d8f" data-toggle="dropdown" aria-haspopup="true"
              aria-expended="false" class="dropdown-toggle btn btn-default">Dropdown <span class="caret"></span>
      </button>
      <ul class="dropdown-menu dropdown-menu-left" role="menu" aria-labelledby="dropdown-144d8f">
        <li class="action" data-js="1">
          <a role="menuitem" tabindex="-1" href="/action">Action</a>
        </li>
        <li class="another-action" data-js="2">
          <a role="menuitem" tabindex="-1" href="/another_action">Another action</a>
        </li>
      </ul>
    </div>

Dropdown button options

Add options after the first two params to customize the dropdown button.

Options:

type (same as button): :primary, :success, :info, :warning, :danger, :link and :default (default).

size (same as button): :large, :small and :xsmall, and a default size would be applied if size option is left blank.

align: :left and :right. By default, a dropdown menu is automatically positioned 100% from the top and along the left side of its parent. Add align: :right to right align the dropdown menu.

direction: Use :up and :down to change the caret's direction.

btn_group_html: Used to hold options for the dropdown's wrapper.

  
    <%= dropdown 'Dropdown',
                 [{type: :link, text: 'Action', link: '/action'},
                 {type: :link, text: 'Another action', link: '/another_action'},
                 {type: :link, text: 'Separated link', link: '/separated_link'}],
                 id:             'dropdown-btn',
                 type:           :success,
                 size:           :large,
                 align:          :right,
                 direction:      :up,
                 class:          'en',
                 data:           {js: 1},
                 btn_group_html: {id: 'btn-group', class: 'zh', data: {js: 2}} %>

generates the following HTML:

EXAMPLE
  • Action
  • Another action
  • Separated link
  
    <div id="btn-group" class="btn-group dropup zh" data-js="2">
      <button name="button" type="button" id="dropdown-btn" data-toggle="dropdown" data-js="1" aria-haspopup="true"
              aria-expended="false" class="dropdown-toggle btn-lg btn btn-success en">Dropdown <span class="caret"></span>
      </button>
      <ul class="dropdown-menu dropdown-menu-right" role="menu" aria-labelledby="dropdown-btn">
        <li><a role="menuitem" tabindex="-1" href="/action">Action</a></li>
        <li><a role="menuitem" tabindex="-1" href="/another_action">Another action</a></li>
        <li><a role="menuitem" tabindex="-1" href="/separated_link">Separated link</a></li>
      </ul>
    </div>
