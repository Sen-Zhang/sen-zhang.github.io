Badge

Use the badge helper to display Bootstrap-styled badge.

Basic badges

Use badge without options to render a basic badge.

  
    <%= badge 21 %>

generates the following HTML:

EXAMPLE
21 
  
    <span class="badge">21</span>

Customize badge tag

Add tag option to change the badge wrapper tag.

  
    <%= badge 21, tag: :em %>

generates the following HTML:

EXAMPLE
21 
  
    <em class="badge">21</em>

Complex badges

You can also specify custom options which will be added to the badge's wrapper tag.

  
    <%= badge 'Hello', tag: :i, id: 'badge', class: :en, data: {js: 1} do %>

generates the following HTML:

EXAMPLE
Hello 
  
    <i id="badge" class="badge en" data-js="1">Hello</i>
