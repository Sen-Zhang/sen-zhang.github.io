Breadcrumb

Use the breadcrumb helper to display Bootstrap-styled breadcrumbs.

Basic breadcrumbs

Use breadcrumb without options to render a basic badge.

  
    <%= breadcrumb [{text: 'Foo', link: '/bar'}, {text: 'Bar', link: '/bar'}, {text: 'Foobar', link: '/foobar'}] %>

generates the following HTML:

EXAMPLE
  1. Foo
  2. Bar
  3. Foobar
  
    <ol class="breadcrumb">
      <li><a href="/foo">Foo</a></li>
      <li><a href="/bar">Bar</a></li>
      <li class="active">Foobar</li>
    </ol>

Customize breadcrumb

Add options to breadcrumb container or to the links inside

  
    <<%= breadcrumb [{text: 'Foo', link: '/bar', class: 'bc-1'},
                     {text: 'Bar', link: '/bar', class: 'bc-2'},
                     {text: 'Foobar', link: '/foobar'}],
                    id: 'breadcrumb', class: :en, data: {js: 1} %>

generates the following HTML:

EXAMPLE
  1. Foo
  2. Bar
  3. Foobar
  
    <ol id="breadcrumb" class="breadcrumb en" data-js="1">
      <li><a class="bc-1" href="/bar">Foo</a></li>
      <li><a class="bc-2" href="/bar">Bar</a></li>
      <li class="active">Foobar</li>
    </ol>
