Button

Use the button helper to display Bootstrap-styled button.

Basic button

Use button without options to render a basic button.

  
    <%= button 'Button' %>

generates the following HTML:

EXAMPLE
  
    <button class="btn btn-default">Button</button>

To include HTML tags or a long text in the button, pass your content in a block.

  
    <%= button do %>
      <span>Hello</span> <em>World</em>!
    <% end %>

generates the following HTML:

EXAMPLE
  
    <button class="btn btn-default">
      <span>Hello</span> <em>World</em>!
    </button>

Contextual button

Set the type option to change the color (and semantic context) of the button.

Available contexts are :primary, :success, :info, :warning, :danger, :link and :default (default).

  
    <%= button 'Primary', type: :primary %>
  
    <%= button 'Success', type: :success %>
  
    <%= button 'Info', type: :info %>
  
    <%= button 'Warning', type: :warning %>
  
    <%= button 'Danger', type: :danger %>
  
    <%= button 'Link', type: :link %>
  
    <%= button 'Default', type: :default %>

generates the following HTML:

EXAMPLE
  
    <button class="btn btn-primary">Primary</button>
    <button class="btn btn-success">Success</button>
    <button class="btn btn-info">Info</button>
    <button class="btn btn-warning">Warning</button>
    <button class="btn btn-danger">Danger</button>
    <button class="btn btn-link">Link</button>
    <button class="btn btn-default">Default</button>

Button Sizes

Set the size option to change the size of the button.

Available sizes are :large, :small and :xsmall, and a default size would be applied if size option is left blank.

  
    <%= button 'Large Button', size: :large %>
  
    <%= button 'Default Button' %>
  
    <%= button 'Small Button', size: :small %>
  
    <%= button 'Extra Small Button', size: :xsmall %>

generates the following HTML:

EXAMPLE
  
    <button class="btn btn-default btn-lg">Large Button</button>
    <button class="btn btn-default">Default Button</button>
    <button class="btn btn-default btn-sm">Small Button</button>
    <button class="btn btn-default btn-xs">Extra Small Button</button>

Block level buttons

Set the layout option to :block to create block level buttons those that span the full with of a parent.

  
    <%= button 'Block level button', type: :success, size: :large, layout: :block %>
  
    <%= button 'Block level button', type: :info, size: :large, layout: :block %>

generates the following HTML:

EXAMPLE
  
    <button class="btn btn-success btn-lg btn-block">Block level button</button>
    <button class="btn btn-info btn-lg btn-block">Block level button</button>

Complex buttons

You can also specify custom options which will be added to the button tag.

  
    <%= button id: 'button', class: :en, data: {js: 1}, type: :warning, size: :large do %>
      <span>Hello</span> <em>World</em>!
    <% end %>

generates the following HTML:

EXAMPLE
  
    <button id="button" class="btn btn-warning btn-lg en" data-js="1">
      <span>Hello</span> <em>World</em>!
    </button>
