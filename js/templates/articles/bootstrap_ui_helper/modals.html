Modal

Use the modal helper to display Bootstrap-styled modal.

Basic modals

Use modal and button_html options to render a basic button and a modal along with modal_header, modal_title, modal_body and modal_footer to render each individual section inside the modal.

To include HTML tags or a long text in the panel, pass your content in a block.

  
    <%= modal do %>
  
      <%= modal_header 'Modal Title' %>
  
      <%= modal_body 'Modal body' %>
  
      <%= modal_footer 'Modal Footer' %>
  
    <% end %>
  
  
    <%= modal do %>
  
      <%= modal_header do %>
  
        <%= modal_title 'Modal Title' %>
  
      <% end %>
  
      <%= modal_body 'Modal body' %>
  
      <%= modal_footer 'Modal Footer' %>
  
    <% end %>
  
  
    <%= modal do %>
  
      <%= modal_header do %>
  
        <em>Modal Heading</em>
  
      <% end %>
  
      <%= modal_body do %>
  
        <em>Modal body</em>
  
      <% end %>
  
      <%= modal_footer do %>
  
        <em>Modal Footer</em>
  
      <% end %>
  
    <% end %>
  
  
    <%= modal do %>
  
      <%= modal_header do %>
  
        <%= modal_title do %>
  
          <em>Modal Title</em>

Button Options

Options for the button are held in button_html and available options are the same as the button helper except option caption.

Options:

caption: text of the button.

size: :large, :small and :xsmall.

type: :primary, :success, :info, :warning, :danger, :link and :default (default).

layout: block.

  
    <%= modal button_html: {caption: 'Modal button', type: :success, size: :large} do %>
  
      <%= modal_header 'Modal Title' %>
  
      <%= modal_body 'Modal body' %>
  
    <% end %>

generates the following HTML:

Modal Title

Modal body

    <div class="modal fade" id="modal-8d1093" tabindex="-1" role="dialog" aria-hidden="true">
      <div class="modal-dialog">
        <div class="modal-content">
          <div class="modal-header">
            <button type="button" class="close" data-dismiss="modal"><span aria-hidden="true">x</span></button>
            <h4 class="modal-title">Modal Title</h4></div>
          <div class="modal-body">Modal body</div>
        </div>
      </div>
    </div>

Modal sizes

Use size option to customize modal size, and available options are :small and :large

  
    <% modal size: :large, button_html: {caption: 'Large modal'} do %>
  
      <%= modal_header 'Modal Title' %>
  
      <%= modal_body 'Modal body' %>
  
    <% end %>
  
  
    <% modal size: :small, button_html: {caption: 'Small modal'} do %>
  
      <%= modal_header 'Modal Title' %>
  
      <%= modal_body 'Modal body' %>
  
    <% end %>

Complex modal container

Options for modal, modal dialog and modal content are also supported.

  
    <%= modal id: 'm', class: 'm', data: {js: 1},
  
              modal_dialog_html: {id: 'md', class: 'd', data: {js: 2}},
  
              modal_content_html: {id: 'mc', class: 'c', data: {js: 3}} do %>
  
      <%= modal_header 'Modal Title' %>
  
      <%= modal_body 'Modal body' %>
  
    <% end %>

generates the following HTML:

Modal Title

Modal body
  
    <button data-toggle="modal" data-target="#m" class="btn btn-default">Modal</button>
    <div id="m" class="modal fade m" data-js="1" tabindex="-1" role="dialog" aria-hidden="true">
      <div id="md" class="modal-dialog d" data-js="2">
        <div id="mc" class="modal-content c" data-js="3">
          <div class="modal-header">
            <button type="button" class="close" data-dismiss="modal"><span aria-hidden="true">x</span></button>
            <h4 class="modal-title">Modal Title</h4></div>
          <div class="modal-body">Modal body</div>
        </div>
      </div>
    </div>

Complex modal sections

Options for modal header, modal title, modal body and modal footer are also supported.

For modal title the default wrapper tag is h4, and you can customize it by using tag option.

  
    <%= modal do %>
  
      <%= modal_header id: 'mh', class: 'd', data: {js: 1} do %>
  
        <%= modal_title id: 'mt', class: 't', data: {js: 2}, tag: :span do %>
  
          <strong>Modal Title</strong><em> - Hello World</em>
  
        <% end %>
  
      <% end %>
  
      <%= modal_body 'Hello World!', id: 'mb', class: 'b', data: {js: 3} %>
  
      <%= modal_footer 'modal footer', id: 'mf', class: 'f', data: {js: 4} %>
  
    <% end %>

generates the following HTML:

Modal Title - Hello World
Hello World!
