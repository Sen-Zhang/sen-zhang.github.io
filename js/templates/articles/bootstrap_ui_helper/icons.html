Icon

Use the icon helper to render font awesome icons.

Basic icon

Use icon with the icon type to render a basic icon.

  
    <%= icon :user %>
  
    <%= icon :plus_square %>

generates the following HTML:

EXAMPLE
  
    <i class="fa fa-user"></i>
    <i class="fa fa-plus-square"></i>

Icon sizes

Customize icon size with size option: :lg, 2x, 3x, 4x and 5x

  
    <%= icon :user, size: :lg %>
  
    <%= icon :user, size: '2x' %>
  
    <%= icon :user, size: '3x' %>
  
    <%= icon :user, size: '4x' %>
  
    <%= icon :user, size: '5x' %>

generates the following HTML:

EXAMPLE
  
    <i class="fa fa-user fa-lg"></i>
    <i class="fa fa-user fa-2x"></i>
    <i class="fa fa-user fa-3x"></i>
    <i class="fa fa-user fa-4x"></i>
    <i class="fa fa-user fa-5x"></i>

Fixed Width Icons

Set fw option to true to render fixed width icons.

  
    <div class="list-group">
  
      <span class="list-group-item">
  
        <%= icon :home, fw: true %> Home
  
      </span>
  
      <span class="list-group-item">
  
        <%= icon :book, fw: true %> Library
  
      </span>
  
      <span class="list-group-item">
  
        <%= icon :pencil, fw: true %> Application
  
      </span>
  
    </div>

generates the following HTML:

EXAMPLE
Home Library Application 
  
    <div class="list-group">
      <span class="list-group-item">
        <i class="fa fa-home fa-fw"></i> Home
      </span>
      <span class="list-group-item">
        <i class="fa fa-book fa-fw"></i> Library
      </span>
      <span class="list-group-item">
        <i class="fa fa-pencil fa-fw"></i> Application
      </span>
    </div>

Bordered icons

Set border option to true to render a bordered icon.

  
    <%= icon :user, border: true %>

generates the following HTML:

EXAMPLE
  
    <i class="fa fa-user fa-border"></i>

Pulled icons

Set pull option to :right or :left to render a pulled icon.

  
    <p>
  
      <%= icon :user, size: '2x', pull: :left, border: true %>
      ... tomorrow we will run faster, stretch out our arms farther...
      And then one fine morning— So we beat on, boats against the current,
      borne back ceaselessly into the past.
    </p>

generates the following HTML:

EXAMPLE

... tomorrow we will run faster, stretch out our arms farther... And then one fine morning— So we beat on, boats against the current, borne back ceaselessly into the past. 

  
    <p>
      <i class="fa fa-user fa-2x fa-border fa-pull-left"></i>
      ... tomorrow we will run faster, stretch out our arms farther...
      And then one fine morning— So we beat on, boats against the current,
      borne back ceaselessly into the past.
    </p>

Animated Icons

Set animate option to :spin or :pulse to render animated icons.

CSS3 animations aren't supported in IE8 - IE9.
  
    <%= icon :spinner, animate: :spin %>
  
    <%= icon :spinner, animate: :pulse %>

generates the following HTML:

EXAMPLE
  
    <i class="fa fa-spinner fa-spin"></i>
    <i class="fa fa-spinner fa-pulse"></i>

Rotated & Flipped Icons

Set orientation option to render animated icons.

Available options are :horizontal, :vertical, '90', '180' and '270'.

  
    <%= icon :shield %> normal
  
    <%= icon :shield, orientation: '90' %> fa-rotate-90
  
    <%= icon :shield, orientation: '180' %> fa-rotate-180
  
    <%= icon :shield, orientation: '270' %> fa-rotate-270
  
    <%= icon :shield, orientation: :horizontal %> fa-rotate-horizontal
  
    <%= icon :shield, orientation: :vertical %> fa-rotate-vertical

generates the following HTML:

EXAMPLE
normal
fa-rotate-90
fa-rotate-180
fa-rotate-270
fa-flip-horizontal
fa-flip-vertical 
  
    <i class="fa fa-shield"></i> normal
    <i class="fa fa-shield fa-rotate-90"></i> fa-rotate-90
    <i class="fa fa-shield fa-rotate-180"></i> fa-rotate-180
    <i class="fa fa-shield fa-rotate-270"></i> fa-rotate-270
    <i class="fa fa-shield fa-flip-horizontal"></i> fa-flip-horizontal
    <i class="fa fa-shield fa-flip-vertical"></i> fa-flip-vertical

List Icons

icon_list and icon_list_item helpers to render list icons.

  
    <%= icon_list do %>
  
      <%= icon_list_item ' Home', icon_html: {type: :home} %>
  
      <%= icon_list_item ' Library', icon_html: {type: :book} %>
  
      <%= icon_list_item ' Settings', icon_html: {type: :cog} %>
  
    <%= end 5>

generates the following HTML:

EXAMPLE
  • Home
  • Library
  • Settings
  
    <ul class="fa-ul">
      <li><i class="fa fa-home"></i> Home</li>
      <li><i class="fa fa-book"></i> Library</li>
      <li><i class="fa fa-cog"></i> Settings</li>
    </ul>

Options for ul, li and icons are also supported.

To include HTML tags or a long text in the icon list item, pass your content in a block.

  
    <%= icon_list id: 'il', class: 'cil', data: {js: 1} do %>
  
      <%= icon_list_item id: 'ili1', class: 'cili', data: {js: 2},
  
                         icon_html: {type: :home, id: 'i1', class: 'i', data: {js: 2}} do %>
  
        <em>Home</em>
  
      <% end %>
  
      <%= icon_list_item id: 'ili2', class: 'cili', data: {js: 3},
  
                         icon_html: {type: :book, id: 'i2', class: 'i', data: {js: 3}} do %>
  
        <em>Library</em>
  
      <% end %>
  
      <%= icon_list_item id: 'ili3', class: 'cili', data: {js: 4},
  
                          icon_html: {type: :cog, id: 'i3', class: 'i', data: {js: 4}} do %>
  
        <em>Settings</em>
  
      <% end %>
  
    <% end %>

generates the following HTML:

EXAMPLE
  • Home
  • Library
  • Settings 
  
    <ul id="il" class="fa-ul cil" data-js="1">
      <li id="ili1" class="cili" data-js="2"><i id="i1" class="fa fa-home i" data-js="2"></i>
        <em>Home</em>
      </li>
      <li id="ili2" class="cili" data-js="3"><i id="i2" class="fa fa-book i" data-js="3"></i>
        <em>Library</em>
      </li>
      <li id="ili3" class="cili" data-js="4"><i id="i3" class="fa fa-cog i" data-js="4"></i>
        <em>Settings</em>
      </li>
