Panel

Use the panel helper to display Bootstrap-styled panel.

Basic panels

Use panel without options to render a basic panel along with panel_heading, panel_title, panel_body and panel_footer to render each individual section inside the panel.

To include HTML tags or a long text in the panel, pass your content in a block.

  
    <%= panel do %>
  
      <%= panel_heading 'Panel Heading' %>
  
      <%= panel_body 'Hello World' %>
  
      <%= panel_footer 'Panel Footer' %>
  
    <% end %>
  
  
    <%= panel do %>
  
      <%= panel_heading do %>
  
        <%= panel_title 'Panel Title' %>
  
        <% end %>
  
      <%= panel_body 'Hello World' %>
  
      <%= panel_footer 'Panel Footer' %>
  
    <% end %>
  
  
    <%= panel do %>
  
      <%= panel_heading do %>
  
        <em>Panel Heading</em>
  
      <% end %>
  
      <%= panel_body do %>
  
        <em>Hello World</em>
  
      <% end %>
  
      <%= panel_footer do %>
  
        <em>Panel Footer</em>
  
      <% end %>
  
    <% end %>
  
  
    <%= panel do %>
  
      <%= panel_heading do %>
  
        <%= panel_title do %>
  
          <em>Panel Title</em>
  
        <% end %>
  
      <% end %>
  
      <%= panel_body do %>
  
        <em>Hello World</em>
  
      <% end %>
  
      <%= panel_footer do %>
  
        <em>Panel Footer</em>

    <% end %>

Contextual panels

Set the type option to change the color (and semantic context) of the panel header and border.

Available contexts are :primary, :success, :info, :warning, :danger and :default.

  
    <%= panel type: :primary do %> ... <% end %>
  
    <%= panel type: :success do %> ... <% end %>
  
    <%= panel type: :info do %> ... <% end %>
  
    <%= panel type: :warning do %> ... <% end %>
  
    <%= panel type: :danger do %> ... <% end %>
  
    <%= panel type: :default do %> ... <% end %>

Customize panel title tag

By default the wrapper tag of panel title is h3, and you can set the tag option to change it. This option is only supported by panel_helper.

  
    <%= panel do %>
  
      <%= panel_heading do %>
  
        <%= panel_title 'Panel Title', tag: :span %>
  
        <em>- Hello World</em>
  
      <% end %>
  
      <%= panel_body 'Hello World!' %>
  
    <% end %>

generates the following HTML:

EXAMPLE
Panel Title - Hello World
Hello World!
  
    <div class="panel panel-default">
      <div class="panel-heading">
        <span class="panel-title">Panel Title</span>
        <em>- Hello World</em>
      </div>
      <div class="panel-body">Hello World!</div>
    </div>

Complex panels

You can specify custom options which will be added to each panel section.

  
    <%= panel id: 'panel', class: 'p', data: {js: 1} do %>
  
      <%= panel_heading id: 'panel_heading', class: 'ph', data: {js: 2} do %>
  
        <%= panel_title id: 'panel_title', class: 'pt', data: {js: 3} do %>
  
          <span>Panel Title</span>
  
        <% end %>
  
      <% end %>
  
  
      <%= panel_body 'Hello World', id: 'panel_body', class: 'pb', data: {js: 4} %>
  
      <%= panel_footer 'panel footer', id: 'panel_footer', class: 'text-center', data: {js: 5} %>
  
    <% end %>

generates the following HTML:

EXAMPLE

Panel Title

Hello World
panel footer
  
    <div id="panel" class="panel panel-default p" data-js="1">
      <div id="panel_heading" class="panel-heading ph" data-js="2">
        <h3 id="panel_title" class="panel-title pt" data-js="3">
          <span>Panel Title</span>
        </h3></div>
      <div id="panel_body" class="panel-body pb" data-js="4">Hello World</div>
      <div id="panel_footer" class="panel-footer text-center" data-js="5">panel footer</div>
    </div>
