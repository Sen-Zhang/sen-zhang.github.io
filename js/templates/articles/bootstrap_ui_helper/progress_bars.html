Progress Bar

Use the progress_bar helper to display Bootstrap-styled progress bar.

Basic progress bar

Use progress_bar with a percentage option to render a basic progress bar.

  
    <%= progress_bar percentage: 60 %>

generates the following HTML:

EXAMPLE
  
    <div class="progress">
      <div class="progress-bar" style="width: 60%" role="progressbar" aria-valuemax="100" aria-valuemin="0" aria-valuenow="60"></div>
    </div>

Progress bar with a label

Use label option to add a label. Available options are true, false and a string.

  
    <%= progress_bar percentage: 60, label: true %>
  
    <%= progress_bar percentage: 60, label: false %>
  
    <%= progress_bar percentage: 60, label: '60 percent' %>

generates the following HTML:

EXAMPLE
60%
60 percent

    </div>

Contextual progress bars

Set the type option to change the color (and semantic context) of the progress bar.

Available types are :success, :info, :warning and :danger.

  
    <%= progress_bar percentage: 20 %>
  
    <%= progress_bar percentage: 30, type: :success %>
  
    <%= progress_bar percentage: 40, type: :info %>
  
    <%= progress_bar percentage: 50, type: :warning %>
  
    <%= progress_bar percentage: 60, type: :danger %>

generates the following HTML:

EXAMPLE
  
    <div class="progress">
      <div class="progress-bar" style="width: 20%" role="progressbar" aria-valuemax="100" aria-valuemin="0" aria-valuenow="20"></div>
    </div>
    <div class="progress">
      <div class="progress-bar progress-bar-success" style="width: 30%" role="progressbar" aria-valuemax="100" aria-valuemin="0" aria-valuenow="30"></div>
    </div>
    <div class="progress">
      <div class="progress-bar progress-bar-info" style="width: 40%" role="progressbar" aria-valuemax="100" aria-valuemin="0" aria-valuenow="40"></div>
    </div>
    <div class="progress">
      <div class="progress-bar progress-bar-warning" style="width: 50%" role="progressbar" aria-valuemax="100" aria-valuemin="0" aria-valuenow="50"></div>
    </div>
    <div class="progress">
      <div class="progress-bar progress-bar-danger" style="width: 60%" role="progressbar" aria-valuemax="100" aria-valuemin="0" aria-valuenow="60"></div>
    </div>

Striped progress bar

Set striped option to true to render a striped progress bar.

  
    <%= progress_bar percentage: 60, striped: true %>

generates the following HTML:

EXAMPLE
  
    <div class="progress">
      <div class="progress-bar-striped progress-bar" style="width: 60%" role="progressbar" aria-valuemax="100" aria-valuemin="0" aria-valuenow="60"></div>
    </div>

Animated progress bar

Set animated option to true to render an animated progress bar.

  
    <%= progress_bar percentage: 60, animated: true %>

generates the following HTML:

EXAMPLE
  
    <div class="progress">
      <div class="progress-bar-striped active progress-bar" style="width: 60%" role="progressbar" aria-valuemax="100" aria-valuemin="0" aria-valuenow="60"></div>
    </div>

Stacked progress bar

Just wrap every progress bar's options hash into an array.

The sum of the percentages of progress bars should not be greater than 100.

  
    <%= progress_bar [{percentage: 20, type: :success},
                      {percentage: 30, striped: true, type: :warning, label: true},
                      {percentage: 40, animated: true, type: :danger, label: '40 percent'}] %>

generates the following HTML:

EXAMPLE
30%
40 percent 
  
    <div class="progress">
      <div class="progress-bar progress-bar-success" style="width: 20%" role="progressbar" aria-valuemax="100"
           aria-valuemin="0" aria-valuenow="20"></div>
      <div class="progress-bar-striped progress-bar progress-bar-warning" style="width: 30%" role="progressbar"
           aria-valuemax="100" aria-valuemin="0" aria-valuenow="30">30%
      </div>
      <div class="progress-bar-striped active progress-bar progress-bar-danger" style="width: 40%" role="progressbar"
           aria-valuemax="100" aria-valuemin="0" aria-valuenow="40">40 percent
      </div>
    </div>
