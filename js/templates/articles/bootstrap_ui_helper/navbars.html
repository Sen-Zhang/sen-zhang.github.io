Navbar

Use the navbar helper to display Bootstrap-styled navbars.

Basic navbars

Use navbar without options to render a basic navbar.

Wrap the content that should always be visible (no matter the screen size) with the vertical helper.

Wrap the content that should be collapsed for small screen sizes with the horizontal helper.

  
    <%= navbar do %>
  
      <%= vertical do %>
  
        <%= navbar_brand 'Brand', '/' %>
  
      <% end %>
  
      <%= horizontal do %>
  
        <%= navbar_menu do %>
  
          <%= navbar_link 'Link', '/link', active: true %>
  
          <%= navbar_dropdown 'Dropdown',
  
                               [{type: :header, text: 'Dropdown header'},
  
                                {type: :link, text: 'Action', link: '/action'},
  
                                {type: :link, text: 'Another action', link: '/another_action'},
  
                                {type: :divider},
  
                                {type: :header, text: 'Another dropdown header'},
  
                                {type: :link, text: 'Separated link', link: '/separated_link'}] %>
  
        <% end %>
  
        <%= navbar_menu position: :right do %>
  
          <%= navbar_link 'Another Link', '/alink' %>
  
          <%= navbar_dropdown 'Another Dropdown',
  
                               [{type: :header, text: 'Dropdown header'},
  
                                {type: :link, text: 'Action', link: '/action'},
  
                                {type: :link, text: 'Another action', link: '/another_action'},
  
                                {type: :divider},
  
                                {type: :header, text: 'Another dropdown header'},
  
                                {type: :link, text: 'Separated link', link: '/separated_link'}] %>
  
          <% end %>
  
        <% end %>
  
      <% end %>

Navbars with inverted colors

Use the :inverse option to invert the palette of colors of the navbar.

  
    <%= navbar inverse: true do %>
  
      <%= vertical do %>
  
        <%= navbar_brand 'Brand', '/' %>
  
      <% end %>
  
    <% end %>

generates a navbar with a black background:

EXAMPLE
Brand 
  
    <nav class="navbar navbar-inverse" data-bui="navbar">
      <div class="container">
        <div class="navbar-header">
          <button class="navbar-toggle collapsed" type="button" data-toggle="collapse" aria-expanded="false">
            <span class="icon-bar"></span>
            <span class="icon-bar"></span>
            <span class="icon-bar"></span>
          </button>
          <a class="navbar-brand" href="/">Brand</a>
        </div>
      </div>
    </nav>

Navbars with a fluid container

Use the :fluid option for a full width navbar, spanning the entire width of your viewport.

  
    <%= navbar fluid: true do %>
  
      <%= vertical do %>
  
        <%= navbar_brand 'Brand', '/' %>
  
      <% end %>
  
    <% end %>

generates a navbar with a fluid container:

EXAMPLE
Brand

            <span class="icon-bar"></span>
          </button>
          <a class="navbar-brand" href="/">Brand</a>

Static navbars

Use the position: :static option to have the navbar scroll away with the page.

  
    <%= navbar position: :static do %>
  
      <%= vertical do %>
  
        <%= navbar_brand 'Brand', '/' %>
  
      <% end %>
  
    <% end %>

generates a navbar that scrolls away with the page:

EXAMPLE
Brand 
  
    <nav class="navbar navbar-default navbar-static-top" data-bui="navbar">
      <div class="container">
        <div class="navbar-header">
          <button class="navbar-toggle collapsed" type="button" data-toggle="collapse" aria-expanded="false">
            <span class="icon-bar"></span>
            <span class="icon-bar"></span>
            <span class="icon-bar"></span>
          </button>
          <a class="navbar-brand" href="/">Brand</a>
        </div>
      </div>
    </nav>

Navbars fixed to top

Use the position: :top option to fix the navbar at the top of the page.

Set the :padding option to specify the padding to leave between the top of the page and the body (defaults to 70px).

  
    <%= navbar position: :top do %>
  
      <%= vertical do %>
  
        <%= navbar_brand 'Brand', '/' %>
  
      <% end %>
  
    <% end %>

generates a navbar fixed to the top with a padding of 70px (click the button to show/hide):

EXAMPLE
Brand 
  
    <style>body {padding-top: 70px}</style>
    <nav class="navbar navbar-default navbar-fixed-top" data-bui="navbar">
      <div class="container">
        <div class="navbar-header">
          <button class="navbar-toggle collapsed" type="button" data-toggle="collapse" aria-expanded="false">
            <span class="icon-bar"></span>
            <span class="icon-bar"></span>
            <span class="icon-bar"></span>
          </button>
          <a class="navbar-brand" href="/">Brand</a>
        </div>
      </div>
    </nav>

Navbars fixed to bottom

Use the position: :bottom option to fix the navbar at the bottom of the page.

Set the :padding option to specify the padding to leave between the bottom of the page and the body (defaults to 70px).

  
    <%= navbar position: :bottom, padding: 100 do %>
  
      <%= vertical do %>
  
        <%= navbar_brand 'Brand', '/' %>
  
      <% end %>
  
    <% end %>

generates a navbar fixed to the bottom with a padding of 100px (click the button to show/hide):

Complex navbars

You can specify a custom id which will be used for the navbar’s collapsable div.

You can also specify custom options in the vertical and horizontal helpers which will be added to their div tags.

  
    <%= navbar id: 'navbar' do %>
  
      <%= vertical id: 'vertical', class: :en, data: {js: 1} do %>
  
        <%= navbar_brand 'Home', '/' %>
  
      <% end %>
  
      <%= horizontal class: :en, data: {js: 2} do %>
  
        <%= navbar_menu position: :right do %>
  
          <%= navbar_link 'Profile', '/profile' %>
  
          <%= navbar_link 'Settings', '/settings' %>
  
        <% end %>
  
      <% end %>
  
    <% end %>

generates a navbar fixed to the bottom with a padding of 100px (click the button to show/hide):

Home
  • Profile
  • Settings
