Label

Use the label_bui helper to display Bootstrap-styled label.

Basic labels

Use label_bui without options to render a basic label.

  
    <%= label_bui 'work in progress ...' %>

generates the following HTML:

EXAMPLE
work in progress ... 
  
    <span class="label label-default">work in progress ...</span>

Contextual label

Set the type option to change the color (and semantic context) of the label.

Available contexts are :primary, :success, :info, :warning, :danger and :default (default).

  
    <%= label_bui 'work in progress ...', type: :primary %>
  
    <%= label_bui 'work in progress ...', type: :success %>
  
    <%= label_bui 'work in progress ...', type: :info %>
  
    <%= label_bui 'work in progress ...', type: :warning %>
  
    <%= label_bui 'work in progress ...', type: :danger %>
  
    <%= label_bui 'work in progress ...', type: :default %>

generates the following HTML:

EXAMPLE
work in progress ... work in progress ... work in progress ... work in progress ... work in progress ... work in progress ... 
  
    <span class="label label-primary">work in progress ...</span>
    <span class="label label-success">work in progress ...</span>
    <span class="label label-info">work in progress ...</span>
    <span class="label label-warning">work in progress ...</span>
    <span class="label label-danger">work in progress ...</span>
    <span class="label label-default">work in progress ...</span>

Customize label wrapper tag

Add tag option to change the label wrapper tag (span by default).

  
    <%= label_bui 'work in progress ...', tag: :em %>

generates the following HTML:

EXAMPLE
work in progress ... 
  
    <em class="label label-default">work in progress ...</em>

Complex labels

You can also specify custom options which will be added to the label wrapper tag.

  
    <%= label_bui 'work in progress ...', type: :success, id: 'label', class: :en, data: {js: 1}%>

generates the following HTML:

EXAMPLE
work in progress ... 
  
    <span id="label" class="label label-success en" data-js="1">work in progress ...</span>
