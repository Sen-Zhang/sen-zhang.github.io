Bootstrap UI Helper Engine

Overview

This is a rails engine to help you easily build bootstrap style UI components with native active view helpers.

Requirement

  • Rails 3.0+
  • Bootstrap 3.0+
  • Font Awesome v4.4.0
  • jQuery 1.7+

Installation

Add this line to your application's Gemfile:

gem 'bootstrap_ui_helper'

And then execute:

$ bundle

Or install it yourself as:

$ gem install bootstrap_ui_helper

In application.js, add the following line:

//= require bootstrap_ui_helper

In application.css, add the following line:

*= require bootstrap_ui_helper
