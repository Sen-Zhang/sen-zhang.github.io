Nav

Use the nav helper to display Bootstrap-styled navs.

Basic navs

Use nav with active option to display the included links as a row of tabs. active option is used to indicate which link is active, and link position index, link id and class are all accepted as the active nav locator.

  
    <%= nav active: 1 do %>
  
      <%= link_to 'Home', '/home' %>
  
      <%= link_to 'Users', '/users' %>
  
      <%= link_to 'Profile', '/profile' %>
  
    <% end %>

generates the following HTML:

EXAMPLE
  • Home
  • Users
  • Profile
  
    <ul class="nav nav-tabs" data-bui="nav" data-active-el-locator="1">
      <li role="presentation"><a href="/home">Home</a></li>
      <li role="presentation" class="active"><a href="/users">Users</a></li>
      <li role="presentation"><a href="/profile">Profile</a></li>
    </ul>

Use link id as the active locator.

  
    <%= nav active: '#profile_url' do %>
  
      <%= link_to 'Home', '/home' %>
  
      <%= link_to 'Users', '/users' %>
  
      <%= link_to 'Profile', '/profile', id: 'profile_url' %>
  
    <% end %>

generates the following HTML:

EXAMPLE
  • Home
  • Users
  • Profile
  
    <ul class="nav nav-tabs" data-bui="nav" data-active-el-locator="#profile_url">
      <li role="presentation"><a href="/home">Home</a></li>
      <li role="presentation"><a href="/users">Users</a></li>
      <li role="presentation" class="active"><a id="profile_url" href="/profile">Profile</a></li>
    </ul>

Navs with justified tabs

Use the layout: :justified option to make tabs occupy the entire width of their parent.

  
    <%= nav active: 0, layout: :justified do %>
  
      <%= link_to 'Home', '/home' %>
  
      <%= link_to 'Users', '/users' %>
  
      <%= link_to 'Profile', '/profile' %>
  
    <% end %>

generates the following HTML:

EXAMPLE
  • Home
  • Users
  • Profile
  
    <ul class="nav nav-tabs nav-justified" data-bui="nav" data-active-el-locator="0">
      <li role="presentation" class="active"><a href="/home">Home</a></li>
      <li role="presentation"><a href="/users">Users</a></li>
      <li role="presentation"><a href="/profile">Profile</a></li>
    </ul>

Navs with pills

Use the as: :pills option to make tabs look like pills.

  
    <%= nav active: 0, as: :pills do %>
  
      <%= link_to 'Home', '/home' %>
  
      <%= link_to 'Users', '/users' %>
  
      <%= link_to 'Profile', '/profile' %>
  
    <% end %>

generates the following HTML:

EXAMPLE
  • Home
  • Users
  • Profile
  
    <ul class="nav nav-pills" data-bui="nav" data-active-el-locator="0">
      <li role="presentation" class="active"><a href="/home">Home</a></li>
      <li role="presentation"><a href="/users">Users</a></li>
      <li role="presentation"><a href="/profile">Profile</a></li>
    </ul>

Navs with stacked pills

Use the layout: :stacked option to make pills vertically stacked.

  
    <%= nav active: 0, as: :pills, layout: :stacked do %>
  
      <%= link_to 'Home', '/home' %>
  
      <%= link_to 'Users', '/users' %>
  
      <%= link_to 'Profile', '/profile' %>
  
    <% end %>

generates the following HTML:

EXAMPLE
  • Home
  • Users
  • Profile
  
    <ul class="nav nav-pills nav-stacked" data-bui="nav" data-active-el-locator="0">
      <li role="presentation" class="active"><a href="/home">Home</a></li>
      <li role="presentation"><a href="/users">Users</a></li>
      <li role="presentation"><a href="/profile">Profile</a></li>
    </ul>

Customize nav wrapper

Use tag option to change the nav's wrapper tag(ul by default).

  
    <%= nav active: 0, tag: :ol do %>
  
      <%= link_to 'Home', '/home' %>
  
      <%= link_to 'Users', '/users' %>
  
      <%= link_to 'Profile', '/profile' %>
  
    <% end %>

generates the following HTML:

EXAMPLE
  1. Home
  2. Users
  3. Profile
  
    <ol class="nav nav-tabs" data-bui="nav" data-active-el-locator="0">
      <li role="presentation" class="active"><a href="/home">Home</a></li>
      <li role="presentation"><a href="/users">Users</a></li>
      <li role="presentation"><a href="/profile">Profile</a></li>
    </ol>

Complex navs

To include HTML tags or a long text in the nav, pass your content in a block. You can also specify custom options which will be added to the navâ€™s ul tag.

  
    <%= nav as: :pills, id: 'nav', class: :en, data: {js: 1} do %>
  
      <li class="active"><a href="/home">Home</a></li>
  
      <%= link_to 'Users', '/users' %>
  
      <%= link_to 'Profile', '/profile' %>
  
    <% end %>

generates the following HTML:

EXAMPLE
  • Home
  • Users
  • Profile
  
    <ul id="nav" class="nav nav-pills en" data-js="1" data-bui="nav" data-active-el-locator="null">
      <li class="active"><a href="/home">Home</a></li>
      <li role="presentation"><a href="/users">Users</a></li>
      <li role="presentation"><a href="/profile">Profile</a></li>
    </ul>
